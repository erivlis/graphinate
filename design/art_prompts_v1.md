# Graphinate 1.0 Art Prompts

This document contains AI image generation prompts for Graphinate's current architecture (Sources -> Engine -> Builders).

## Prompt Template

Use this structure to generate consistent layouts across different art styles.

```text
"A [Adjective] infographic poster explaining the architecture of the 'Graphinate' Python library in the style of [Artist/Style].

Layout: Left-to-Right flow.
- Section 1 (Left): 'Sources'. Icons representing APIs, Databases, and Files. Data flowing into a funnel.
- Section 2 (Center): 'The Engine'. A central, glowing core labeled 'GraphModel'. Inside, show Python generator functions transforming data into Nodes and Edges.
- Section 3 (Right): 'Builders'. Three distinct outputs branching out: 1. A NetworkX graph structure. 2. A GraphQL logo labeled "GraphQL API". 3. A 3D Visualization (force-directed graph).

Composition: Left-to-Right flow.
- Left (Sources): [Style-specific depiction of raw data, APIs, and Databases] flowing into a central point.
- Center (The Engine): [Style-specific depiction of the 'GraphModel' core] transforming the raw inputs.
- Right (Builders): [Style-specific depiction of the 'Builders'] expressing their outputs.

Style: [Specific technique, medium, line work, lighting].
Color Palette: [Specific colors].
Vibe: [Mood/Atmosphere]."
Typography: [Font style, if applicable].
```
